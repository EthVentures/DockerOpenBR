DockerOpenBR
============
Dockerized implementation of OpenBR, a communal biometrics framework.

Usage
-----

docker run avelkoski/openbr:latest bash -c "br -algorithm AgeEstimation -enroll /images/bush1.jpeg meta.csv &> /dev/null && cat meta.csv"
